# wisdom-tree

An interactive installation where collective knowledge grows into something beautiful.


## 📖 About

Wisdom Tree is a collaborative art installation where visitors contribute their wisdom, advice, or meaningful questions. Each contribution becomes a leaf on a growing tree, with branches forming based on thematic connections. Over time, the tree evolves into a beautiful visualization of collective human insight.

## ✨ Features

- **Organic Growth**: Each contribution adds a new leaf and branch to the tree
- **AI Theme Detection**: Automatically categorizes wisdom into themes (perseverance, courage, kindness, etc.)
- **Interactive Exploration**: Click leaves to read accumulated wisdom
- **Real-time Visualization**: Animated tree with sparkles and smooth transitions
- **Theme Analytics**: Track emerging patterns in collective wisdom
